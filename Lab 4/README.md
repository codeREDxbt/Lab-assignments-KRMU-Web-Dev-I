# Lab 4 - JavaScript Quiz Game

A simple JavaScript quiz game that runs entirely in the browser console, featuring predefined questions, user input via `prompt()`, immediate feedback with `alert()`, and score tracking.

## Features
- Array of quiz questions and answers
- For loop to iterate through questions
- Input normalization with `toLowerCase()` and `trim()`
- Conditional checking and score updating
- Final score display

## How to Run
1. Open your browser's developer console (F12).
2. Copy and paste the contents of `quiz.js`.
3. Press Enter to start the quiz.
4. Answer each prompt and view feedback.

## Questions
- What is the capital of France? (Paris)
- What is 5 * 6? (30)
- Who wrote Romeo and Juliet? (Shakespeare)
- What is the largest planet in our solar system? (Jupiter)
- What color is the sky on a clear day? (Blue)
