## Lab 2 - Personal Portfolio with CSS
Enhanced portfolio with external CSS stylesheet, featuring modern design, responsive layout, and interactive elements.

**Location**: `Lab 2/index.html`

### Features
- External CSS with consistent color theme and Google Fonts (Poppins)
- Sticky navigation bar with hover effects
- CSS box model for proper spacing and layout
- Styled skills table with alternating rows
- Contact form with validation
- Back-to-top button with smooth scrolling
- Professional card-based project layout

## Usage
- **Lab 1**: Navigate to `Lab 1/` folder and open `Portfolio.html` in your browser
- **Lab 2**: Navigate to `Lab 2/` folder and open `index.html` in your browser
