## Lab 2 - Personal Portfolio with CSS
Enhanced portfolio with external CSS stylesheet, featuring modern design, responsive layout, and interactive elements.

**Location**: `Lab 2/index.html`

### Features
- External CSS with consistent color theme and Google Fonts (Poppins)
- Sticky navigation bar with hover effects
- CSS box model for proper spacing and layout
- Styled skills table with alternating rows
- Contact form with validation
- Back-to-top button with smooth scrolling
- Professional card-based project layout

## Lab 3 - Advanced Portfolio (Dark Glassmorphism)
Modern portfolio built with semantic HTML5 and advanced CSS (Flexbox/Grid) featuring a dark neon aesthetic with glassmorphism and motion.

**Location**: `Lab 3/index.html`

### Highlights
- Dark cyberpunk theme with neon accents (`--primary: #00f2ff`, `--secondary: #7000ff`)
- Glassmorphism cards and header using `backdrop-filter: blur(...)`
- Flexbox for header/hero and CSS Grid for skills/projects
- Responsive layout with mobile-first breakpoints
- Animated entrances and interactive hovers (glow, float, slide)
- Brand aligned (“VV” top-left) and footer updated to `codeRED`
- Social links wired: GitHub, LinkedIn, and X (Twitter)
