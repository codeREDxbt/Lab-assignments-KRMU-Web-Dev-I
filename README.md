## Capstone Project - LoveLotus Food Delivery App
A fully responsive food delivery web application built with semantic HTML5 and modern CSS3 (Flexbox/Grid), featuring glassmorphism design and a unique electric purple & teal color palette.

**Location**: `Capstone/index.html`

### Theme & Design
- **Brand**: LoveLotus with custom SVG logo
- **Color Palette**: Electric Purple (`#6C5CE7`) & Robin's Egg Teal (`#00CEC9`) on Cool Grey
- **Visual Style**: Glassmorphism with backdrop blur effects and vibrant gradient backgrounds

### Features
- Semantic HTML5 structure (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- Sticky glassmorphic navigation with logo and cart icon
- Hero section with call-to-action and hero image
- Search bar with glass effect above categories
- Category cards with hover animations
- Featured menu section with sidebar filters (price, dietary)
- Restaurant showcase featuring top Delhi NCR restaurants (Indian Accent, Bukhara, Olive Bar & Kitchen)
- Full contact form with location, phone, and email info
- Responsive design (Desktop ≥1024px, Tablet ~768px, Mobile ≤480px)

### Layout Techniques
- CSS Flexbox for header, navigation, and hero alignment
- CSS Grid for categories, menu items, and restaurant cards
- CSS Variables (`:root`) for consistent theming
- Media queries for responsive breakpoints
- Hover effects and bouncy transitions
- High-quality food images from Unsplash
